二、详细安装教程

1. 获取插件安装包

您可以通过以下三种正规渠道获取插件：
- 小米应用商店搜索"科学上网插件"
- 访问小米官网下载专区
- 主流浏览器扩展商店（如Chrome Web Store）

重要提示：务必从官方渠道下载，避免安装被篡改的恶意版本。

2. 分步安装指南

对于Chrome浏览器用户：

  1. 打开浏览器输入chrome://extensions/
  2. 启用右上角的"开发者模式"
  3. 将下载的.crx文件拖入窗口
  4. 确认安装提示

对于小米浏览器用户：

  1. 进入设置→扩展管理
  2. 点击"从文件安装"
  3. 选择下载的安装包
  4. 授予必要权限

安装后检查：成功安装后，浏览器右上角会出现蓝色盾牌图标，表示插件已就绪。

三、深度配置详解

1. 服务器优化设置

点击插件图标进入主界面后，您将看到：

  • 智能模式：自动选择延迟最低的服务器（推荐新手使用）
  • 手动模式：可自由选择特定国家/地区的节点
    • 亚洲节点：适合访问中日韩网站（延迟通常<100ms）
    • 欧美节点：适合访问国际流媒体（推荐法兰克福、洛杉矶节点）
    • 特殊节点：部分提供P2P支持

专业技巧：长按节点可进行ping测试，选择响应时间最短的服务器。

2. 协议选择指南

插件支持多种VPN协议，各有特点：

|----------|------|--------|----------|

建议：日常浏览选择OpenVPN，需要高速传输时切换至WireGuard。

3. 高级功能配置

在设置→高级选项中，您可以：

  • 启用分流模式：仅对特定网站使用VPN
  • 设置自动连接：当访问被屏蔽网站时自动启用
  • 配置DNS设置：使用8.8.8.8等公共DNS提升解析速度
  • 开启流量压缩：节省移动数据用量

FAQ

Potatso Lite 能否兼容最新的 iOS 系统？
Potatso Lite 更新频率较低，在新版本 iOS 上可能出现兼容性问题。若遇到无法连接或闪退情况，建议升级至其他活跃维护的客户端如 Shadowrocket 或 Quantumult X。
Clash 的策略组支持哪些节点选择策略？
支持按延迟选择、随机选择、轮询或手动选择节点。结合规则匹配，用户可实现不同网站或应用使用不同节点，提高访问速度和网络稳定性，同时降低管理复杂度。
PassWall2 在路由器上使用 TUN 模式会有什么影响？
启用 TUN 模式会拦截并转发路由器上全部或选定流量，这会增加路由器 CPU 与内存负担，可能影响路由器其他服务表现。建议在性能足够的设备上启用，或仅对特定设备/端口进行分流。
如何在路由器上使用 PassWall2 为不同设备设置不同策略？
在 PassWall2 的规则或策略组中为固定 IP 或 MAC 地址创建特定路由条目，将其映射到不同的策略组（如工作机走直连、游戏机走低延迟节点），并保存规则后重载以生效。
Surfboard 与 Clash 有什么关系？
Surfboard 是一款基于 Clash 核心的 iOS 应用，专为苹果用户设计。它通过友好的界面实现分流管理、节点切换和延迟测试等功能。与 Clash 类似，Surfboard 也支持订阅更新和多种代理协议，兼容性非常好。
Mellow 与 Clash 的核心区别是什么？
Mellow 采用基于透明代理的架构，适合路由或系统级流量重定向，而 Clash 更偏向规则式应用代理。Mellow 的灵活性更高，但配置复杂；Clash 则兼顾易用性与功能扩展性。
Shadowrocket 的自动切换功能如何配置？
Shadowrocket 支持根据策略组和节点延迟自动切换最优节点。用户可设置测速频率和延迟阈值，程序会定期测试节点并自动切换，以保证网络稳定性和速度。
SSR 的协议参数和混淆参数如何配置最安全？
选择 AEAD 或协议插件可提升安全性，混淆方式如 tls1.2_ticket_auth 可以模拟正常 HTTPS 流量。合理搭配协议和混淆参数，能够有效抵御 DPI 检测和流量封锁。
V2Ray 的 WebSocket + TLS 如何防止流量被检测？
WebSocket + TLS 可将 V2Ray 流量伪装成正常 HTTPS 请求，隐藏真实数据特征，配合域名伪装和策略组使用，可有效绕过封锁，提高节点安全性和连接稳定性。
SSR 的加密方式推荐使用哪种？
SSR 支持多种加密方式，如 aes-256-cfb、chacha20、rc4-md5 等。建议优先选择 chacha20 系列算法，在移动设备上性能更佳，同时具有较强的安全性。
